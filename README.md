
## CONFIGURACIÓN DE MONITORES JBL EON
- AUX 1: Monitor mezcla para baterista (drums + algo de voz)
- AUX 2: Monitor mezcla para guitarrista/tecladista (su instrumento + voces)
- AUX 3: Monitor mezcla para vocalista (voces + algo de instrumentos)
- AUX 4: Monitor spare o mezcla general

## NOTAS TÉCNICAS
- Utilizar escenas en la Midas M32R para guardar ajustes de cada banda
- Aprovechar el kit Telefunken para la batería (prioridad)
- Usar los clamps de pinza para los OH y algunos micrófonos cercanos
- Preparar 4 bases con boom (voces y amplificador) y 2 bases rectas (hi-hat y spare)
- Usar multicontacto Furman para alimentación limpia
- Marcar con cinta de Gaffer las posiciones en el tapete de batería
- Configurar 4 buses principales en la M32R para submixes (drums, guitars, vocals, keys)
- Aplicar compresión, EQ y Gates desde los procesadores internos de la M32R

## LISTA DE CONEXIONES DE CABLES
1. 10 cables XLR para conectar micrófonos a la consola
2. 5 cables TS para conexiones de instrumentos a DI
3. 2 cables TRS para conexiones estéreo
4. Cables Telefunken del kit para batería
5. Extensiones eléctricas para alimentación de equipos
